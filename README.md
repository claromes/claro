- [icons8.com](https://icons8.com/)
- [svgtopng.com](https://svgtopng.com/)
- [codebeautify.org/image-to-base64-converter](https://codebeautify.org/image-to-base64-converter) (to ASCII)
- [urlencoder.org](https://www.urlencoder.org/)
